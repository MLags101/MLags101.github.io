Custom Remote

About

This project was a personal exploration of mine into RF modules and PCB fabrication. I wanted to have a device that would be able to selectively activate any device which could be easily outfitted with a matching module, or by a MOSFET switch controlled by the RF module. I was able to explore various libraries for RF communication and I enjoyed testing out the RF module hardware. I decided it was important to make the resulting circuit as compact as possible which meant it needed to be put onto a PCB. The opportunity to fabricate my own PCB was an important component of this project as it was my responsibility as Electronics Apprentice to make a procedure for a new PCB mill acquired by the Invention Studio. I applied my experience with KiCAD and learned how to use FlatCAM so I could make this project as a demo board for the new mill.
