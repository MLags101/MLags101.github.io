Electronics Lockbox

About

At the Invention Studio, a makerspace on campus, Prototyping Instructors (PIs) have the opportunity to specialize in a specific tool group and aid more in the care and operation of that group. This role is called a master, and the steps to become one varies from tool groups, but each requires a comprehensive project that demonstrates the PI's knowledge in that area as well as serves a useful purpose to that tool group. I chose to make a lockbox for the electronics cabinet keys. This box is packed full of fun features, including an RFID sensor that works with student ID cards, a MOSFET driven solenoid for the lock mechanism, an RF module for remote opening, and a spark gap for audio deterence.
