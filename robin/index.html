Robin

About

Named after the nephew of Kermit, Robin is a smaller drone designed as a learning tool for BetaFlight and more risky control schemes. Like Kermit(the drone), Robin's frame is completely custom designed and 3D printed. It uses an F7 flight controller-ESC stack bought from AliExpress and a Raspberry Pi Zero W 2 as a flight computer. Because the F7 runs BetaFlight, I couldn't use the same MAVProxy code I wrote for Kermit. Instead, I learned how to use the MultiWii Serial protocol to send direct RC channel values via the Pi. I'm currently working on a handheld remote control to communicate with the Pi and control the RC channels based on buttons for throttle and yaw, as well as the remote gyro-state for pitch and roll control.
